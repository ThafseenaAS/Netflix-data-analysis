# Netflix-data-analysis
Netflix data analysis project using Python, Pandas, and Matplotlib

## Objective
To analyze Netflix movies and TV shows dataset and find insights using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Process
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization

## Key Insights
- Movies are more than TV shows
- United States has the highest content
- Netflix content incresed significantly over the years
- Drama is most common genre

## Conclusion
This project helps understand Netflix content trends using data analysis.
